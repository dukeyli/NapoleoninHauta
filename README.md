## NapoleoninHauta

#### Aihemäärittely

Kyseessä on pasianssi, jossa käydään läpi pakkaa ja pyritään pinoamaan 
kortit viiteen pinoon siten, että yhdessä pinossa on neljä kertaa 
kortit 6, 5, 4, 3, 2 ja ässä ja neljässä muussa pinossa on kortit 7-13.
